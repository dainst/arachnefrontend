Über Arachne Interfaces

Um höchste Daten-Interoperabilität zu gewährleisten, stellt die Arachne Datenbank ihre Daten in mehreren Metadatenformaten und durch verschiedene Anwendungsschnittstellen zur Verfügung.

OAI-PMH

Die Basis-URI der OAI-PMH-Schnittstelle ist http://arachne.uni-koeln.de/OAI-PMH/oai-pmh.xml. Ein Beispiel finden Sie hier. Beachten Sie jedoch, dass wir derzeit an einer neuen Schnittstelle für den Datenexport arbeiten.

Das OAI-PMH-Protokoll wird in dem Open Archives Protocol beschrieben.

Bücher via OAI-PMH

Der Inhalt des Arachne Buch-Browsers kann über eine weitere OAI-PMH-Schnittstelle (basierend auf jOAI) im Format METS/MODS abgerufen werden.

Die Basis-URI der Schnittstelle ist http://arachne.uni-koeln.de/oai/provider. Ein Beispiel finden Sie hier.

Weitere Spezifikationen finden Sie unter:

http://www.loc.gov/standards/mets/

http://www.loc.gov/standards/mods/

http://dfg-viewer.de/profil-der-metadaten/
