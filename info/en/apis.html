About Arachne Interfaces

In order to guarantee the highest data interoperability, the Arachne database provides its data in several metadata formats delivered through different application interfaces.

Books via OAI-PMH

The OAI-PMH interface is specified in the Open Archives Protocol. The contents of the Arachne bookbrowser can be harvested in the METS/MODS format via a OAI-PMH interface (based on jOAI).

The base URI of this interface is http://arachne.uni-koeln.de/oai/provider. An example can be found here.

For specifications see:

http://www.loc.gov/standards/mets/

http://www.loc.gov/standards/mods/

http://dfg-viewer.de/profil-der-metadaten/
